## 🚀 部署指南

### Vercel部署（推荐）
```bash
# 安装Vercel CLI
npm i -g vercel

# 登录Vercel
vercel login

# 部署到Vercel
vercel --prod

# 或使用pnpm
pnpm vercel --prod
```

### 环境变量配置
创建 `.env.local` 文件：
```env
NEXT_PUBLIC_SITE_URL=https://mahavatar-narsimha.com
NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX
```
